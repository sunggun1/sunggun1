안녕하세요! 저는 5년차 백엔드 개발자입니다.

•비동기 백엔드 시스템에 대한 전문성을 보유하고 있습니다. Netty, Spring WebFlux, Kafka, CompletableFuture등을 활용해 비동기 아키텍처를 설계·구현했으며, 안정적인 이벤트 처리를 위해 재시도 메커니즘과 Dead Letter Queue 기반의 처리 구조까지 구축해 본 경험이 있습니다.

•시스템 성능 최적화에도 강점이 있습니다. MongoDB 복합 인덱스를 적용해 조회 응답 시간을 5초에서 1.5초 수준으로 단축, MySQL 응답 시간 3초에서 2초로 단축 등 실제 운영 환경에서 체감 가능한 성능 개선을 이끌어냈습니다.

•DevOps와 클라우드 CI/CD 자동화 경험을 갖추고 있습니다. Docker, Jenkins, Harbor, Kubernetes, ArgoCD 등을 활용해 CI/CD 파이프라인과 고가용성 환경을 구축하고, 자동 배포와 확장 가능한 운영 환경을 구현해 서비스 안정성과 운영 효율을 높였습니다.
